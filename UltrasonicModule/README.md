# Ultrasonic Sensor
---
This is project meant for me to learn about embedded software and systems.

## Description
---
As stated above, I put together this device to learn about embedded systems. The project was created using the STM32Cube ecosystem. I used both STM32CubeMX to configure the microcontroller and STM32CubeIDE to write the software. The device can measure distances up to 450cm according to the datasheet.

## Materials
---
- STM32 Nucleo-F303K8 MCU
- RCWL-1601 Ultrasonic Module
- Misc. components (breadboard, wires, etc.)
